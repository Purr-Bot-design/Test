Commands

The Bot has some awesome commands you might want to explore. They all are free to use but don't forget to try them all. The bot is still in development so there is a slight chance that you might face a problem in something or the other. Below are the list of commands and their uses! The prefix of the bot is (prefix)

General Commands

Name
Uses
help
shows all the available commands available in the bot
user <:member:>
shows the user info that you have mentioned while entering the command
dm <:member:>
the bot dms the user that you mentioned with a message which you entered in the syntax
ping
tells the latency of the bot
logs
shows the recent changes and fixes in the bot. Mainly, its a list of updates and the new commands
greet
greets the user with a heartwarming message
info
shows the current stats and information regarding the bot
cat
gives out a wholesome image of a kitten or a cat
pfp
shows the current profile picture of the user you mentioned or your profile picture if the user is not mentioned
serverinfo
gives out certain things about the server in which you entered this command.
feedback
a way of giving a feedback to the development of the bot and its working. The feedback you will give will reach to the developer.
echo <:message:>
this command will make the bot say whatever you typed the message in the syntax
dog
this command spawns out a cute dog for you!

Calculation Commands

Name
Uses
add <:number1:> <:number2:>
sums up the number you entered in the syntax
sub <:number1:> <:number2:>
subtracts the numbers you entered in the syntax
multi <:number1:> <:number2:>
gives out the product that you entered in the syntax
div <:dividend:> <:divisor:>
divides the given number that you entered according to the syntax
facto <:number:>
gives the factorial series of a number that you entered in the syntax
sq
gives out the answer of the square number that you entered

Moderation Commands

Name
Uses
kick <:member:>
command which kicks out the member mentioned if the bot is given the permission to kick members
ban <:member:>
bans the member which is mentioned in the syntax from the server
clear <:number:>
this command is used to clear a number of messages at once.
boom <:channel:>
the command deletes the channel that is mentioned above in the syntax
rules
command prints out the preset embeded set of rules made by the bot. The embed message would be print in that channel where the command is ran
unban <:member:>
this command is the contrary of the kick command. The tag of the member which has to be unbanned should be in full form (e.g. @abcd#1234)
poll
holds out a poll, the options are only two right now. Rest would be added in future
